# PROJECT BRIEF

## Project Overview
**Project Name:** Transcription App (Echos)  
**Platform:** Flutter (Cross-platform)  
**Primary Language:** Dart  

## Current State
- Existing Flutter application with transcription functionality
- Multiple audio transcription services (Vosk, Whisper)
- Session-based recording and transcription management
- Offline-first architecture with local storage

## Project Architecture
- **Frontend:** Flutter widgets and screens
- **State Management:** Provider pattern
- **Local Storage:** Encrypted storage for transcriptions
- **Audio Processing:** Vosk (offline) and Whisper (AI) services

## Key Features
- Real-time transcription during recording
- Session management and organization
- Offline transcription capabilities
- Cross-platform compatibility (iOS, Android)

## Technology Stack
- **Framework:** Flutter (Dart)
- **State Management:** Provider
- **Storage:** Encrypted local storage
- **Audio:** Native audio recording
- **AI Models:** Vosk (offline), Whisper (cloud)

## Development Status
- Mature codebase with established architecture
- Active development and maintenance
- Ready for enhancements and new features
