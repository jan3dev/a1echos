# PRODUCT CONTEXT

## Product Overview
**Product Name:** Echos - Transcription App  
**Domain:** Audio Transcription & Session Management  
**Platform:** Mobile (iOS & Android)  
**Target Users:** Professionals, Students, Content Creators  

## User Experience
### Primary User Journey
1. **Session Creation** - User initiates a new recording session
2. **Real-time Recording** - Audio captured with live transcription
3. **Transcription Processing** - Multiple AI models process audio
4. **Review & Edit** - User reviews and manages transcriptions
5. **Organization** - Sessions organized and stored locally

### Key User Interactions
- **Recording Controls** - Start/stop/pause recording
- **Live Transcription View** - Real-time text display
- **Session Management** - Browse, organize, and manage sessions
- **Selection & Export** - Select text portions for sharing
- **Settings Configuration** - Adjust transcription preferences

## Product Features
### Core Features
- **Offline Transcription** - Vosk-based local processing
- **AI Transcription** - Whisper integration for enhanced accuracy
- **Session Organization** - Hierarchical session management
- **Real-time Display** - Live transcription during recording
- **Cross-platform** - iOS and Android compatibility

### Advanced Features
- **Encrypted Storage** - Secure local data protection
- **Audio Visualization** - Waveform display during recording
- **Transcription Selection** - Granular text selection tools
- **Export Options** - Multiple output formats
- **Offline-first** - Full functionality without internet

## Business Value
- **Privacy-first** - All processing can be done locally
- **Accessibility** - Transcription for hearing impaired users
- **Productivity** - Rapid conversion of audio to text
- **Flexibility** - Multiple transcription engine options
- **Reliability** - Offline capabilities ensure availability

## User Personas
### Primary: Professional Users
- Journalists, lawyers, researchers
- Need accurate, private transcription
- Value offline capabilities and data security

### Secondary: Students
- Lecture recording and note-taking
- Cost-sensitive, need reliable transcription
- Benefit from real-time processing

### Tertiary: Content Creators
- Podcast/video transcription needs
- Quality and editing features important
- Export and sharing capabilities valued
