# Product Context

## Product Name
**DolphinEcho** - Local AI Transcription App

## Product Vision
DolphinEcho empowers users with private, offline transcription capabilities by leveraging local AI models. The app provides a seamless experience for recording and transcribing audio without compromising user privacy or requiring internet connectivity.

## Target Users
- **Privacy-conscious professionals** who need transcription without cloud dependency
- **Journalists and researchers** requiring offline transcription capabilities
- **Students and academics** for lecture and interview transcription
- **Content creators** needing quick transcription for video/podcast content
- **Business professionals** for meeting notes and voice memos

## Core Value Propositions
1. **Privacy First**: All transcription happens locally - no data leaves the device
2. **Offline Capability**: Full functionality without internet connection
3. **Dual Model Support**: Choose between real-time (Vosk) and high-accuracy (Whisper) transcription
4. **Session Organization**: Organize transcriptions into logical groups
5. **Professional Quality**: Material Design 3 interface with modern UX patterns

## User Journey
1. **Setup**: Install app and initialize transcription models
2. **Session Creation**: Create or select a session for organizing transcriptions
3. **Model Selection**: Choose between Vosk (real-time) or Whisper (high-accuracy)
4. **Recording**: Record audio with visual feedback and waveform display
5. **Transcription**: Automatic transcription based on selected model
6. **Management**: Review, edit, and organize transcriptions within sessions
7. **Export/Share**: Access transcribed text for further use

## Competitive Advantages
- **Local Processing**: No subscription fees or cloud dependencies
- **Model Flexibility**: Switch between speed-optimized and accuracy-optimized models
- **Session Management**: Better organization than simple list-based apps
- **Open Source Models**: Uses established Vosk and Whisper technologies
- **Cross-Platform**: Single codebase for iOS and Android

## Success Metrics
- User retention and engagement with transcription features
- Accuracy satisfaction with both Vosk and Whisper models
- Session organization adoption rate
- App performance and battery efficiency
- User privacy and security satisfaction

## Key Use Cases
1. **Recording and Transcribing Conversations**
   - Record in-person meetings or conversations
   - Generate accurate transcriptions locally
   - Save and organize transcriptions

2. **Note Taking**
   - Record spoken notes or ideas
   - Transcribe and review later
   - Export transcriptions to other formats or applications

3. **Accessibility**
   - Convert audio content to text for users with hearing impairments
   - Create searchable text from audio recordings

## User Experience Goals
- Simple, intuitive interface
- Quick access to recording functionality
- Clear organization of recordings and transcriptions
- Easy configuration of transcription settings
- Reliable transcription quality

## Core Features
- Audio recording with adjustable quality settings
- Local transcription using either Vosk or Whisper models
- Session management for organizing recordings
- Transcription review and editing
- Export functionality for sharing transcriptions
